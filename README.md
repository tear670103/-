# Gundam-Supreme-Battle-Pay-Calculator
# 鋼彈爭鋒對決 課金計算器  
1$是真實貨幣的1塊錢，如果是其他國家的，改成商城-儲值-鑽石購買 1塊錢能買多少鑽石數量。  
490NT=1960鑽石、990NT=3960鑽石 是最划算的(1:4)。  
銷售價格那邊，鈔票圖示是銷售價格，點擊鈔票圖示可以換成鑽石銷售價格。  
原本鑽石價格初始值都是補給鑽石售價，金哈囉原本鑽石價格則是商城-商店-每月特惠的金哈囉銷售的鑽石價格。
